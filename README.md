## Overview
This project is a Parking Management System developed in C++ using Object-Oriented Programming (OOP) concepts. It helps manage vehicle entries, exits, parking records, and charges efficiently.

## Features
- Vehicle Entry and Exit Management
- Parking Slot Allocation
- Record Storage using File Handling
- Charge Calculation
- Exception Handling
- Object-Oriented Design

## Technologies Used
- C++
- OOP Concepts
- File Handling

## OOP Concepts Implemented
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
